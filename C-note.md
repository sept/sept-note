##指针
只有它能存储数据地址

所有定义 int * 就是指针变量

定义之后才能把地址赋给它

通过 * 取出该地址内的内容

    int b;
    int a;
    int *ptr;  "定义指针ptr"
    ptr = &a;  "取a 的地址给ptr"

    b = a;	  
    即 b = *ptr
